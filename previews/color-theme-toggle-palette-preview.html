Pinterest Palette Study

One-tap color themes for OOTODAY, same structure, different mood.

这页先只看“配色切换按钮”的主题方向，不改结构，不改排版，不改信息架构。 目标是让你现在这套 editorial 界面能一键换 mood，但仍然保持同一个产品系统。

我这次参考的是 Pinterest 上能访问到的配色 pin / board 标题，以及 Pinterest 官方创意指南里强调的 “从一开始就考虑 palette”和“用 contrast 建立移动端效果”。

参考来源： Black and Chartreuse Color Palette、 Brand App Mobile UI Design Hex Colors Palette、 Mobile App Color Palette、 Pinterest Path to Performance Guide。 Pinterest 对页面抓取限制很强，所以这里是基于可访问 pin 标题和官方指南做的风格提案，不是逐像素复刻。
